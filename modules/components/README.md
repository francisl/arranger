# Data Portal Components

## Build Process
- `cd modules/components`
- `npm i`
- `npm run watch`
- `npm run test`
- Update version in `modules/components/package.json`
- `npm publish`
- `npm i` to sync `package-lock.json`
- You should see on npmjs.com a newly published version
